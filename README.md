# 图库与提示词管理

一个基于浏览器的纯前端图库与提示词管理工具，无需后端服务，所有数据存储在浏览器本地（localStorage）。

## 功能特点

- **卡片盒管理** — 以"卡片盒"形式组织图片，支持单图/多图导入
- **批量导入** — 批量选取图片自动创建卡片盒，自动从 PNG 文件中提取提示词
- **标签系统** — 单选、交集、并集三种标签过滤模式，灵活筛选
- **提示词整理** — 内置提示词清洗与修复工具
- **数据可移植** — 支持 JSON 格式导入/导出，数据完全由你掌控
- **深色/浅色主题** — 适配不同使用环境
- **NSFW 过滤** — 安全模式开关
- **完全离线可用** — 单 HTML 文件，打开即用

## 使用方法

1. 直接用浏览器打开 `图库管理.html`
2. 通过侧边栏导入图片或数据
3. 使用标签、搜索、排序功能管理你的图库

## 技术栈

- 纯 HTML + CSS + JavaScript
- 浏览器本地存储（localStorage）
- 无第三方依赖
